# Network Intrusion Detection System (IDS)

## Objective
To build a Python-based Network IDS that monitors live traffic and detects port scanning behavior.

## Features
- Live packet capture using Scapy
- Port scan detection based on threshold logic
- Alert generation with timestamps
- Persistent alert logging

## Tech Stack
- Python
- Scapy
- Network packet analysis

## How to Run
Run with administrator/root privileges:
```bash
python ids.py
